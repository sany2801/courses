Install:
	1. Download and extract lotus_standard.zip
	2. Right click desired resolution "[...].inf" in folder
	3. Click "install"
	4. Navigate to Settings>Devices>Mouse>Additional mouse options>Pointers
	5. Select the desired Scheme from drop down menu
	6. Hit "OK"

Extras:
	- simply backup the original and rename the extra to the original's name and place it
	  in the folder with the rest of the cursors (i.e. "link(1).cur" --> "link.cur")
